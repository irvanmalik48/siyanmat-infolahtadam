## Quickstart

Install the required NPM packages:

```plaintext
pnpm install
```

Run the app using:

```plaintext
pnpm dev
```

or

```plaintext
pnpm run dev
```

To build the app, run this command:

```plaintext
pnpm build
```

or

```plaintext
pnpm run build
```
